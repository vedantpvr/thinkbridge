CSS Keywords

color

Sets the text color of an element.

background-color

Sets the background color of an element.

margin

Defines space outside the element’s border.

padding

Defines space inside the element, around content.

border

Sets the border style, width, and color.

font-size

Controls the size of text.

display

Specifies how an element is displayed (block, inline, flex, etc.).
