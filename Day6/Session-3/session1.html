HTML Keywords

<html>

Defines the root of an HTML document.

<head>

Contains metadata, title, links, and scripts.

<body>

Defines the content of the document (visible to users).

<h1> to <h6>

Headings from largest (<h1>) to smallest (<h6>).

<p>

Paragraph tag for text content.

<a>

Defines hyperlinks to navigate between pages.

<ul> <ol> <li>

Unordered (ul) and ordered (ol) lists with list items (li).

<div>

Generic container for grouping elements.

<span>

Inline container for text or elements.

<img>

Embeds images in the document.

<link>

Links external resources like CSS files.

<script>

Embeds or references JavaScript code.
