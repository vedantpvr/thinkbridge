JavaScript Keywords

var

Declares a variable (function-scoped, older way).

let

Declares a block-scoped variable (modern way).

const

Declares a constant (value cannot be reassigned).

function

Defines a reusable block of code.

return

Specifies the value a function returns.

if

Executes code conditionally based on a true/false value.

for

Loop that runs a block of code multiple times.

while

Loop that runs while a condition is true.

switch

Executes code based on different cases.

try/catch

Handles errors in code execution.

class

Defines a blueprint for creating objects.

new

Creates an instance of a class.
